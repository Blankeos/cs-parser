## Get started

> Make sure to install [.Net SDK](https://dotnet.microsoft.com/en-us/download) if you haven't yet

1. Clone this repo and change your directory

```sh
git clone <repolink>
cd parser-csharp
```

2. Run the project in your terminal

```
dotnet run
```
